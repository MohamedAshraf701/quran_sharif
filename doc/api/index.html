quran_sharif

The quran_sharif package is a Flutter library that provides the Quran in both English and Arabic translations. It is designed to be easily integrated into Flutter applications, allowing users to access and interact with the Quranic text for study, reference, or recitation purposes.

Features

  • Complete text of the Quran in English and Arabic
  • Verse-by-verse navigation and search functionality
  • Support for switching between different translations
  • Transliteration of Arabic verses for non-Arabic speakers
  • Beautifully formatted display with adjustable font sizes
  • Bookmarking and saving favorite verses

Installation

To use the quran_sharif package, add it as a dependency in your pubspec.yaml file:

dependencies:
  quran_sharif: <latest_version>
